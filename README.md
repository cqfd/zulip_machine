# ZulipMachine

EventMachine bindings for Zulip's API.

## Installation

Add this line to your application's Gemfile:

    gem 'zulip_machine'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zulip_machine

## Usage

Take a look at the `examples` directory.
